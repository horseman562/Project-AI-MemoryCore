## Design Rules (Non-Negotiable)
1. Each restaurant must use a DIFFERENT layout strategy — never repeat the same hero/menu pattern
2. Each restaurant must have a DIFFERENT color scheme — no two sites should look related
3. Always use CSS custom properties (design tokens) — never hardcode colors
4. Always include grain texture, fluid typography, and hover glow — these are the quality baseline
5. Mobile responsive — all sections must work on phone

## Pitch Rules
1. Always in Malay (Bahasa Malaysia), casual/friendly tone
2. Never say "free demo" — it's a demo to earn the paid project
3. Always include demo link + portfolio link (bayuratech.com)
4. Send to multiple channels — IG first, then WhatsApp, then Email
5. Walk-in is last resort — friendly, not pushy
